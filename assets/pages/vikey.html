ViKey Entry

Version 1.0 is written in Python and leverages an existing Northeastern system to open the room door. Each user is given a six digit pin which when entered into the physical keypad outside the door, uses Northeastern's online door control system to send a signal to the door's lock. Additional features include entry music selected from a list of predefined song segments, doorbell, and entry logging.
Version 2.0 will be written in NodeJS and AngularJS. Functionality will be translated into an API for increased flexibility and customization. Pin entry will no longer be limited to the physical keypad. Options such as NFC tags, smartphone home screen shortcuts, and web browser entry can easily be implemented.
