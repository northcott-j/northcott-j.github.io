Facebook Analyzer

Background

Original project was started by @Lackoftactics and I became a contributor. The goal was to create a way to analyze the zip file from Facebook and get info about friends, ranking by message, vocabulary, contacts, friends added statistics and more.

Features

I refactored the original project to make it easier to add new analyses to the process. I created mixins and abstract classes to reduce the duplication of logic. I also did a performance benchmark and made code adjustments to speed up the analysis. The final output is an Excel file, but there has been progress towards exporting a shareable, dynamic, html site.

Technical Architecture

Written using Ruby with gems Nokogiri, parallel, and axlsx to assist in processing.
