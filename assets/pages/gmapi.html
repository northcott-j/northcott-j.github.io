GMAPI Visualization

This tool was written using AngularJS, NodeJS, Node Webkit, Python, and Google Maps API. Using a spreadsheet or database, aggregate values are visualized on a map using a color gradient. Various filters can be applied depending on the information to manipulate the scope of the data. The backend information is loaded into memory at the launch of the application, so changes happen almost instantaneously. Thanks to Node Webkit, this tool can be run on Linux, Windows and MacOS using the same code.
